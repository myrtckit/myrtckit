# myrtckit
myrtckit是由Zlmediakit和metaRTC联手打造的全栈webRTC解决方案，可快速建立视频会议/直播互动/AI等多种音视频应用。

# New Feature
## Sfu(zlm)
  * 集成mqtt等信令服务
  * 服务端会议录制
  * 语音室多路音频合流混音
  * 嵌入式系统深度优化
  * Docker/Kubernetes一键启动
## Client(metaRTC)
  * AEC/AGC/ANS/VAD/CNG等多种音频处理
  * 桌面/文档远程控制与共享
  * windows/linux/mac桌面sdk和demo
  * android/ios原生sdk和demo
  * flutter/reactNative/uniApp/weex等sdk和demo
## Network
  * UDP/TCP/TURN支持和自适应切换
  * nack/fec/svc/pacer等更多的弱网策略助力更强大弱网自适应能力
